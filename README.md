RestaurantDepo
==============
Features:

    1. Allow customers to see foods served by a restaurant
    2. Allow customers to add to their orders and see their total cost.

Tools:

    1. JQuery Mobile
    2. AngularJS
    3. Google Map API
    4. Yelp API
    5. JQuery
    
Flatform:

    1. Web Browser
    2. Android
    3. IOs
